### Quick Start
#### open terminal

Download `git clone https://github.com/roeunapireak/smart-notes-app.git`

checking python package version
`pip --version`

install pyqt5 module
`pip3 install pyqt5`

to run application
`python3 {data dir}/smart-notes-app/releases/{version}/smart-note-app.py`
